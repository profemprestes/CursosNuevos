Descripción General del Curso

Este curso está diseñado para estudiantes de tercer año de Educación Media Superior con especialidad en Tecnologías de la Información. Aprenderás las metodologías, técnicas y herramientas fundamentales de la ingeniería del software para desarrollar aplicaciones de calidad. El enfoque práctico te permitirá aplicar los conocimientos en proyectos reales, desde la especificación de requerimientos hasta la gestión y planificación de proyectos completos.

Unidad 1: Especificación de Requerimientos

Aprende a identificar, documentar y validar los requisitos de un sistema software.

  • Introducción a la ingeniería de requisitos
  • Técnicas de elicitación de requisitos
  • Clasificación de requisitos funcionales y no funcionales
  • Documentación de requisitos (SRS)
  • Validación y verificación de requisitos
  • Gestión de cambios en los requisitos

En esta unidad, aprenderás las mejores prácticas para capturar correctamente las necesidades del cliente y transformarlas en requisitos claros, completos y verificables. Realizarás entrevistas, cuestionarios y otras técnicas para obtener información de calidad, y aprenderás a documentarla siguiendo estándares profesionales.

Unidad 2: Modelado de Sistemas

Diseño y representación de sistemas software mediante diferentes técnicas de modelado.

  • Introducción al modelado de software
  • Lenguaje Unificado de Modelado (UML)
  • Diagramas de casos de uso
  • Diagramas de clases y objetos
  • Diagramas de secuencia y actividad
  • Patrones de diseño software

Esta unidad te proporcionará las herramientas para representar visualmente la estructura y comportamiento de los sistemas software. Aprenderás a utilizar UML para comunicar diseños de manera efectiva, documentando tanto los aspectos estáticos como dinámicos de un sistema, y aplicarás patrones de diseño para resolver problemas comunes.

Ir a Unidad 2

Unidad 3: Gestión, Planificación y Control de Proyecto

Metodologías y técnicas para la correcta administración de proyectos de software.

  • Ciclos de vida del desarrollo de software
  • Metodologías ágiles vs. tradicionales
  • Planificación y estimación de proyectos
  • Gestión de riesgos en proyectos software
  • Control de calidad y testing
  • Gestión de la configuración y herramientas

En la unidad final, aprenderás a gestionar proyectos software de manera efectiva, aplicando metodologías ágiles como Scrum y herramientas de gestión de proyectos. Estudiarás técnicas de estimación, planificación y seguimiento, así como estrategias para mitigar riesgos y asegurar la calidad del producto final.
